# Boggle Wiz
A program that generates and solves a game of boggle

- [Run](#run)
- [Test](#test)
- [Design](#design)


## Run
Run the following from the `./boggle` directory
```
python main.py
```

## Test
Run the following from the `./boggle` directory
```
python -m unittest tests.board tests.round tests.trie
```
